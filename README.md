## Library Management
#### Build on Material 3 UI

## Features
- [x] Add Book
- [x] SignUp/SignIn seperated for both Admin and User
- [x] Issue Book to User

## Tech Stack
- [x] Kotlin
- [x] Google Material 3
- [x] Google Firebase

## Note
- [x] This is a simple library management app.
